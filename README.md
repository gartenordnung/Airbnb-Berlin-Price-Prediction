# Airbnb-Berlin-Price-Prediction
Predict the Price of Airbnb listings in Berlin for university <br>
Main directory location contains mostly work with TF-IDF and so on. 
RidgePCA folder contains work on separate version of the data with most columns and one-hot-encoded amenities. <br>
Presentations/ slideshows are in the folder of the respective relevant notebooks and are in .key format for powerpoint. <br>
More work from us can be found under: https://github.com/gartenordnung/Project-AIRBNB-Berlin-Price-Prediction/tree/main , but would take too much time during the project presentation.
